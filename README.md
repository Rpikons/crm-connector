Project Title: Notion CRM Connector
Description:
This project is an automated API connector that captures leads from a web form and saves them instantly into a Notion database. It is a lightweight "headless" solution designed to eliminate manual data entry and centralize contact management.

Key Features:

Direct Integration: Seamless communication with the official Notion API.

Security: API keys and sensitive data managed via Environment Variables.

Cloud Ready: Fully configured for immediate deployment on platforms like Railway or Heroku.

Responsive Frontend: Clean and functional UI built with HTML5 and CSS3.

Tech Stack:

Backend: Node.js, Express.js.

Database: Notion (via Notion SDK/Axios).

Deployment: GitHub & Railway.
